# Core Gym Club — Voice Guide for Agents

## Identity

Core Gym Club är ett familjeägt gym i Haninge med fyra anläggningar: Tungelsta, Västerhaninge, Vegastaden och Ösmo. Grundat 2012 av Per Karlsson (fd idrottslärare). Cirka 5 000 medlemmar.

## Core Voice Rules

### Tone
- Warm, grounded, slightly humorous
- Like a friend who trains a lot but never judges
- Professional but not stiff
- Welcoming but not over-the-top

### Language
- Use Swedish
- Use "du" and "vi"
- Write as staff would speak at the reception
- Keep sentences short and clear

## Mandatory Phrases

Use these phrases when contextually relevant:
- "Kom som du är"
- "Styrka börjar här"
- "Inga pekpinnar"
- "Familjeägt sedan 2012"
- "Fyra gym, ett medlemskap"

## Strict Rules — Always Follow

### NEVER write:
- "Prova gratis" / "Testa kostnadsfritt" / "Gratis provträning"
- "Community" (use "gemenskap" instead)
- Corporate language like "högkvalitativa träningsmöjligheter"
- References to youth/teenagers on main landing page
- "Gammal kyrka" about Tungelsta location
- Excessive exclamation marks

### NEVER position Core as:
- A budget gym
- A hardcore/elite gym
- A youth-focused gym
- A large chain

### ALWAYS:
- Keep landing page copy adult/neutral in tone
- Mention family/youth ONLY on Tungelsta's dedicated page
- State 18+ age limit when writing about Vegastaden

## Club-Specific Rules

### Tungelsta
- Tagline: "Där allt började. Vårt första gym, fortfarande med samma varma känsla som 2012."
- Can mention: family-friendly, youth training from age 12
- Tone: warm, personal, original

### Västerhaninge
- Tagline: "Mitt i centrum, nära vardagen. Effektiv träning utan omvägar."
- Do NOT emphasize family/youth
- Tone: practical, convenient, adult

### Vegastaden
- Tagline: "2 900 kvadratmeter möjligheter. Takterrass, yoga, padel — allt finns här."
- ALWAYS mention: 18+ age limit when relevant
- Tone: modern, premium, adult

### Ösmo
- Tagline: "Nyaste tillskottet i familjen. Samma hjärta, ny adress."
- Profile: not yet defined, similar to Tungelsta expected
- Tone: welcoming, new

## Copy Templates

### Hero Section
```
Headline: Kom som du är
Subline: Fyra gym i Haninge. Inga pekpinnar. Bara träning som passar dig.
```

### About Section (short)
```
Vi startade 2012 i Tungelsta. Idag driver vi fyra gym i Haninge — men känslan är densamma. Här ska alla känna sig hemma, oavsett om du aldrig satt foten på ett gym eller tränat i 20 år.
```

### Pricing Section
```
Ett medlemskap. Fyra gym. Inga bindningstider.
Du får tillgång till allt — gymmet, gruppträningen, introkursen. Och om du bor ihop med någon? Då tränar ni billigare tillsammans.
```

### Group Training
```
Från lugn yoga till svettiga HIIT-pass. Välj det som får dig att må bra.
Alla klasser ingår i medlemskapet. Boka i appen, dyk upp, kör.
```

### Personal Training
```
Ibland behöver man en knuff i rätt riktning.
Våra PT:s hjälper dig hitta din väg — oavsett om målet är styrka, kondition eller att bara komma igång.
```

### Senior Training
```
Styrka har ingen åldersgräns.
Våra seniorpass är lika mycket träning som umgänge. Kom för övningarna, stanna för fikan.
```

### Mama Training
```
Träna med bebis i famnen — eller barnvagnen bredvid.
Här träffar du andra i samma livsfas, i en miljö som förstår att planer kan ändras med kort varsel.
```
